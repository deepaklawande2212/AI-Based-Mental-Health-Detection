# AI Based Mental Health Detection Via Social Media Analysis

**Short description**  
This project analyzes social media posts to detect early signs of mental health issues using NLP and machine learning models.

## Project objective
Build an automated system that ingests social media text, preprocesses it, performs sentiment/emotion analysis and classification, and visualizes results for further investigation.

## Technologies used
- TypeScript (frontend)
- Node.js (backend)
- Python (ML and NLP models)
- HTML / CSS
- Vite, Tailwind (if used)
- Any ML libs: TensorFlow / PyTorch / scikit-learn (as applicable)
